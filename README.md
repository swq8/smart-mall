## ![favicon](https://alex.g686.net/favicon-32x32.png) alex商城系统 

#### 项目介绍
   alex商城系统基于SpringBoot构建，程序包含大量测试用例确保每次调整发布时均能保持稳健；
   在确保软件质量的同时对速度追求极致，客户端流畅度及压力测试数据在同类软件中表现优秀。
   无论开箱即用还是二次开发，alex商城系统都将是您在企业级商城系统中的优秀选择。
